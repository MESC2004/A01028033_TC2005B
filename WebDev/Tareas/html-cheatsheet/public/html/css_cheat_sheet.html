CSS Cheat Sheet

border

Sets the width, style, and color of the border of an element. In this case, a 2px solid dark blue border.

border-left, border-right, border-top, border-bottom

Sets the width, style, and color of the left, right, top, and bottom borders of an element, respectively. Allows for cool things like flag or rainbow borders!

border-radius

Sets the radius of the element's corners. Allows for curved edges.

background-image

Sets the background image of an element. FERRETS!!!!

color

Sets the color of the text. Not much to say here.

font

Sets all the font properties in one declaration. In this case Italic Bold Serif.

font-family

Specifies the font family for the text.

font-style, font-weight

Specifies the font style (italic, normal, etc.) and the weight (boldness) of the font.

width, height

Sets the width and height of an element.

list-style

Specifies the type of list-item marker in a list.

margin

Sets the margin area on all four sides of an element.

padding

Sets the padding area on all four sides of an element.

overflow

Specifies how to handle content that overflows the element's box.

display

Specifies the display behavior of an element.

position

Specifies the type of positioning method used for an element (static, relative, absolute, fixed, sticky).

filter

Applies graphical effects like blur or color shift to an element.
