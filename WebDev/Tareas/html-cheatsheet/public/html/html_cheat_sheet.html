Basic Tags:

These are the basic tags that are used in HTML

1. a - Anchor / Hyperlinks

The anchor tag is used to create hyperlinks. It is used with the href attribute to specify the URL of the page that the link goes to. In this case, it takes you to google.com.

In code this looks like this:

<p>The anchor tag is used to create hyperlinks. It is used with the href attribute to specify the <a href="https.gogle.com">URL</a> of the page that the link goes to. In this case, it takes you to goole.com</p>

2. p - Paragraph

The paragraph tag is used to create paragraphs of text. It is used to separate blocks of text from each other. In this case, it is used to separate the text above from the text below.

In code this looks like this:

<p>The paragraph tag is used to create paragraphs of text. It is used to separate blocks of text from each other. In this case, it is used to separate the text above from the text below.</p>

4. div - Content Division

A div helps to divide or group HTML elements together.

It is used to group elements together so that they can be styled with CSS, or modified through other methods.

<p>It is used to group elements together so that they can be styled with CSS, or modified through other methods.</p>
</div>
